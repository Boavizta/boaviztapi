# API

Source code of Boavizta's API

## Technos

* Flask 2.0.2
* Python >= 3.6

## Coding convention

[PEP8](https://www.python.org/dev/peps/pep-0008/)

## Project first Install & run
```python3 -m venv venv``` 

```pip3 install -r requirements.txt``` 

```flask run```
