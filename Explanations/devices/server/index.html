    Bottom-up approach

    Usage impact

    Both power consumption and consumption profile are implemented.

    Consumption profile

    A server consumption profile is of the form :

    \[ \text{CP}_{\text{component}} = \text{consumption_profile}_{\text{component}} \]
    \[ \text{CP}(\text{workload}) = (\text{CP}_{\text{CPU}}(\text{workload}) + \text{CP}_{\text{RAM}}(\text{workload})) * (1 + \text{other_consumption_ratio}) \]

    \(\text{CP}_{\text{CPU}}(\text{workload})\) and \(\text{CP}_{\text{RAM}}(\text{workload})\) depend on the technical characteristics of the RAM and CPU. \(\text{other_consumption_ratio}\) is used to account for the electrical consumption of the other components (other than RAM and CPU).
