    Terminals & Peripherals

    General description

    Terminals & peripherals may have several types (typically pro and perso).

    Name Unit Default value Description Example
    type None None A subcategory of device perso

    Embedded impacts

    Most embedded impacts are taken from the Base IMPACTS® ADEME database. When the data comes from another source, it is specified.

    Criteria Implemented Source
    gwp yes Base IMPACTS® ADEME
    adp yes Base IMPACTS® ADEME
